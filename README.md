## 📌 Project Overview

The **General Health Query Chatbot** is an AI-powered conversational assistant that answers general health-related questions using a **Large Language Model (LLM)**.

Built with **Python**, **Hugging Face Transformers**, and **Qwen2.5-0.5B-Instruct**, the chatbot provides quick, clear, and user-friendly responses without requiring any paid API services.

This project demonstrates **Prompt Engineering**, **Natural Language Processing (NLP)**, and **Local LLM Inference**, making it an excellent beginner-friendly AI/ML project.

You: What are the symptoms of fever?

• Headache
• Muscle aches
• Fatigue

---

# 🧠 Prompt Engineering

The chatbot is guided using carefully designed prompts that instruct the model to:

- Answer only health-related questions
- Provide short and understandable responses
- Avoid diagnosing diseases
- Recommend consulting a healthcare professional for serious symptoms
- Maintain a polite and helpful conversational style
